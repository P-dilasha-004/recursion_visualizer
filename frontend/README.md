# Recursion Step Visualizer

A simple React app to visualize recursive functions step by step. It helps students and developers understand recursion by showing the call stack and explanations for each recursive call.

## Features

- Visualize Factorial and Fibonacci recursion
- Enter any custom recursive function
- Step through recursion with a dynamic call stack view
- Display the function code alongside the visualization

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed  
- npm (comes with Node.js)

git clone recursion_visualizer
cd frontend
